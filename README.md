# Love is Foundation Static Website Prototype

A simple static website prototype using HTML, CSS, and JavaScript.

## Files

- `index.html` - Main website structure
- `styles.css` - Design, layout, colors, responsiveness
- `script.js` - Mobile menu, event rendering, reveal animations, prototype form
- `README.md` - Setup notes

## How to preview locally

Open `index.html` in a browser.

Or use VS Code Live Server.

## How to host on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files.
3. Go to repository **Settings**.
4. Go to **Pages**.
5. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.
7. GitHub will provide a live website link.

## Important notes before using for the real site

This is a front-end prototype only.

The contact form does not send real messages. Connect it to one of these later:

- Google Forms
- Formspree
- Netlify Forms
- A custom backend

For real production use, also check:

- Accessibility
- Mobile layout
- Final brand copy
- SEO metadata
- Real events
- Real images
- Privacy policy
- Cookie policy if tracking is added
