# Love is Foundation — Living Logo Scroll Site

A responsive static website built around the official seven-circle Love is Foundation logo and Apple-style scroll storytelling.

## Run locally

```bash
python -m http.server 8080
```

Open `http://localhost:8080/lif-living-logo-site/` if serving from `/mnt/data`, or `http://localhost:8080/` if serving inside this folder.

## Pages
- Home
- About
- Seven Aspects
- What We Do
- Resources
- Join Us
- Contact

## Notes
- The navbar uses the uploaded LiF logo.
- The animated emblem is an SVG interpretation that resolves into the actual logo image.
- The self-discovery choice is saved in localStorage.
- The contact form is a front-end demonstration and needs a real endpoint before launch.
- Reduced-motion preferences are respected.
