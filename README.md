# Love is Foundation Multi-Page Static Website Prototype

This version expands the prototype into a multi-page website

## Pages

- `index.html` - Homepage
- `about.html` - Mission, values, story
- `programs.html` - What LiF offers
- `garden.html` - Interactive LiF Garden Playground
- `community.html` - Engagement platform flow
- `resources.html` - Filterable resource library
- `events.html` - Events listing prototype
- `volunteer.html` - Volunteer pathways
- `contact.html` - Contact form and FAQ

## Interactions

- Mouse glow
- Hero card tilt
- Clickable LiF Garden
- Soft click sounds
- Tabbed offerings
- Resource filters
- Event show more/show fewer
- FAQ accordions
- Visitor pathway matching tool

## Hosting on GitHub Pages

1. Upload all files and the `assets` folder to a GitHub repository.
2. Go to Settings → Pages.
3. Select Deploy from a branch.
4. Choose `main` and `/root`.
5. Save.
