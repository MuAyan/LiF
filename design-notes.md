# V3 Design Notes

## Main change

The website has been expanded from a one-page prototype into a multi-page website so LiF's offerings can be properly organized.

## Why

A one-page site is useful for a quick pitch, but LiF needs room for:
- organization story
- values and principles
- programs
- LiF Garden Playground
- engagement platform
- events
- resources
- volunteer pathways
- contact and onboarding

## Current page structure

Home → high-level overview and routing  
About → mission, values, story, process  
What We Offer → programs and service areas  
LiF Garden → interactive community metaphor  
Community → engagement platform flow  
Resources → filterable content library  
Events → calendar-style listing  
Volunteer → participation pathways  
Contact → form, social links, FAQ  

## Logo / brand direction

Uses a simplified LiF flower mark inspired by the proposed logo redesign direction. The goal is to keep the symbolism while making it cleaner and more usable across pages.
